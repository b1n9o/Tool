# 文件平铺移动脚本使用说明

## 脚本概述

这是一个用于将指定目录下所有文件（包括子目录中的文件）移动到新目录并保持平铺结构的Shell脚本。脚本会自动跳过.DS_Store文件，并处理文件名冲突问题。

## 功能特点

📁 **目录平铺**：将多层目录结构中的文件移动到单一目录中  
🔄 **文件名冲突处理**：自动为同名文件生成唯一文件名  
🚫 **跳过系统文件**：自动跳过macOS的.DS_Store文件  
⚡ **高效处理**：使用find命令高效遍历文件  
🔧 **灵活配置**：可通过变量自定义默认目标文件夹名称  
🛡️ **安全检查**：防止目标目录已存在导致的数据风险
